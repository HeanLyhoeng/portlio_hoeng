# Client Testimonial Images

This folder should contain 20 client testimonial images.

## Image Naming Convention

Please name your images as follows:
- `client-1.jpg`
- `client-2.jpg`
- `client-3.jpg`
- ... (continue through)
- `client-20.jpg`

## Image Requirements

### Web Browser View
- **Width**: 1024px to 2500px
- **Aspect Ratio**: 3:1 (width:height)
- **Pixel Resolution**: 72 ppi
- **Recommended Size**: 1200px × 400px (optimal balance)
- **Format**: JPG, PNG, or WebP
- **File Size**: Optimize for web (recommended: under 500KB each)

### Mobile View
- **Minimum Width**: 800px
- **Aspect Ratio**: 4:3 (width:height)
- **Pixel Resolution**: 72 ppi
- **Recommended Sizes**:
  - 800px × 600px (minimum)
  - 1200px × 900px (optimal)
- **Alternative Ratios**: 1:9, 1:1, or 4:5 (for mobile banners)
- **Format**: JPG, PNG, or WebP
- **File Size**: Optimize for mobile (recommended: under 300KB each)

### Performance Tips
- For best performance, use 1200×630 pixels or smaller
- Larger images may impact mobile performance
- Consider using WebP format for better compression
- Optimize images before uploading

## Adding Images

1. Place your 20 client testimonial images in this folder
2. Name them according to the convention above (client-1.jpg through client-20.jpg)
3. The carousel will automatically display them, rotating every 4 seconds
4. Images will automatically adapt to web and mobile viewports

## Note

If images are not found, the carousel will fall back to placeholder images automatically.
